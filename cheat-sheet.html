Anchors (Links Dummy!!!)

Anchor tatgs create links to other pages. The URL goes in the 'href' Attribute Go to DuckDuckGo for more information.

Settin'target="_blank" will open the link in a new window. See MDN for more details.

Anchor tags can also create links to a new location on the same page. Instead of poutting a URL in the "href" attribute, just use the 'id' of the element you would like to navigate to. Jump to the top of the page.

Text Emphasis

Italis

The 'em' element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the 'strong' element to emphasize text. This shows up as bold text in most browsers.

Organizing content with 'div' and 'span'

'div'

The 'div' element is a block element, so each 'div' will show up on a new line.
